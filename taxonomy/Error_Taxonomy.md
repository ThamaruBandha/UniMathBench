# Error Taxonomy (E1–E6)

| Code | Error Category | Description | Example |
|------|----------------|-------------|---------|
| **E1** | Conceptual Error | Wrong concept or theorem. | Uses Bayes' theorem when conditional probability is required. |
| **E2** | Method Error | Incorrect procedure or substitution. | Applies integration by parts instead of substitution. |
| **E3** | Calculation Error | Arithmetic or numerical mistake. | Correct formula but computes 4 × 7 as 24. |
| **E4** | Reasoning Error | Flawed logical explanation. | Gives inconsistent justification despite a correct formula. |
| **E5** | Incomplete Solution | Stops before obtaining the final answer. | Solves part of the problem but omits the final numerical result. |
| **E6** | Hallucination | Fabricated information or unsupported assumptions. | Invents a theorem or assumes values not given in the question. |
